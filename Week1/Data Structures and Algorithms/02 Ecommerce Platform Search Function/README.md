# Exercise 2: E-commerce Platform Search Function

## Objective

Implement Linear Search and Binary Search to find a product in an e-commerce platform using its product ID.

## Implementation

The program stores product details such as product ID, product name, and category. It performs:

- **Linear Search** to search through the products one by one.
- **Binary Search** to search efficiently after sorting the products by product ID.

## Time Complexity

### Linear Search
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)

### Binary Search
- **Time Complexity:** O(log n)
- **Space Complexity:** O(1)

## Conclusion

Linear Search is suitable for small or unsorted datasets. Binary Search is much faster for large datasets but requires the products to be sorted before searching.
