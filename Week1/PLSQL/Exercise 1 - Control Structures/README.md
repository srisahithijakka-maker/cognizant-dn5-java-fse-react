# Exercise 1: Control Structures

## Objective
Implement PL/SQL control structures to automate common banking operations using loops, conditional statements, and DBMS_OUTPUT.

## Scenarios Implemented

### Scenario 1
Applied a 1% reduction in loan interest rates for customers who are above 60 years of age.

### Scenario 2
Updated the `IsVIP` status for customers whose account balance is greater than 10,000.

### Scenario 3
Displayed reminder messages for customers whose loans are due within the next 30 days.

## Concepts Used
- PL/SQL Blocks
- FOR LOOP
- UPDATE Statement
- Conditional Processing
- DBMS_OUTPUT
- COMMIT

## Result
All scenarios executed successfully and produced the expected output.
