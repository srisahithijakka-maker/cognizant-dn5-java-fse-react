# Exercise 3: Stored Procedures

## Objective
Implement PL/SQL stored procedures to automate common banking operations.

## Scenarios Implemented

### Scenario 1
Created a stored procedure to apply 1% monthly interest to all savings accounts.

### Scenario 2
Created a stored procedure to update employee salaries for a selected department by applying a bonus percentage.

### Scenario 3
Created a stored procedure to transfer funds between two accounts after validating the available balance.

## Concepts Used
- Stored Procedures
- Procedure Parameters
- UPDATE Statements
- IF-ELSE Conditions
- COMMIT
- DBMS_OUTPUT

## Result
All stored procedures executed successfully and performed the required banking operations.

## Conclusion
This exercise demonstrates the use of PL/SQL stored procedures to encapsulate business logic, perform database updates, and improve code reusability.
