# Exercise 1 - Logging Error Messages and Warning Levels

## Objective
Demonstrate logging of error and warning messages using the SLF4J logging framework with Logback.

## Technologies Used
- Java
- Maven
- SLF4J
- Logback

## Files
- LoggingExample.java
- pom.xml

## Output
The application logs:
- An ERROR message
- A WARN message

This demonstrates basic logging functionality using the SLF4J framework.
