# Exercise 4: Creating and Configuring a Maven Project

## Objective
Create a Maven project and configure the required Spring Framework dependencies and Maven Compiler Plugin.

## Implementation
- Created a Maven project named **LibraryManagement**.
- Added the following Spring dependencies:
  - Spring Context
  - Spring AOP
  - Spring Web MVC
- Configured the Maven Compiler Plugin to use Java 1.8.
- Reloaded the Maven project to download and resolve all dependencies successfully.

## Outcome
The project is successfully configured with Maven and the required Spring Framework libraries, making it ready for Spring application development.
