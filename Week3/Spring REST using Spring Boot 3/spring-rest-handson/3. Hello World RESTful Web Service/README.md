# Hello World RESTful Web Service

## Objective
Create a RESTful web service using Spring Boot that returns **"Hello World!!"** when the `/hello` endpoint is accessed.

## Technologies Used
- Java 21
- Spring Boot
- Spring Web
- Maven
- SLF4J Logging

## Endpoint
GET /hello

## Output
When the application is running and the endpoint is accessed:
http://localhost:8083/hello

Response:
Hello World!!

The application also logs START and END messages in the console.

## Result
Successfully created and tested a Hello World RESTful Web Service using Spring Boot.
