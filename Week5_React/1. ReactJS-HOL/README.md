# Hands-on 1: Create First React Application

## Objective
Create a React application named **myfirstreact** and display the heading **"Welcome to the first session of React"** on the home page.

---
## Technologies Used
- React.js
- JavaScript
- Node.js
- npm
- Create React App
---

## Steps Performed
1. Created a new React application using Create React App.
2. Opened the project in the IDE.
3. Modified the `App.js` file.
4. Displayed the required heading.
5. Executed the application using `npm start`.
---

## Commands Used
```bash
npx create-react-app myfirstreact
cd myfirstreact
npm start
```
---
## Output
The application runs successfully on:
```
http://localhost:3000
```
and displays:
```
Welcome to the first session of React
```
---

## Result
Successfully created and executed the first React application using Create React App and displayed the required heading.
